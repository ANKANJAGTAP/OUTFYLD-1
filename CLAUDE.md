# OutFyld Design System — "Night Match"

## Non-negotiable bans
- NO gradient hero bands, NO pill badges above headings
- NO icon-in-rounded-square card grids
- NO centered symmetric section layouts as default
- NO Inter/Poppins/Roboto, no emoji as UI elements
- NO default Google Maps styling, no generic stat strips

## Required
- Fonts: [display: condensed sports-style] + [UI: grotesk]
- Dark floodlit palette: charcoal-green base, floodlight 
  white/lime accents, grain overlay on hero sections
- Diagonal pitch-line dividers as the layout motif
- Asymmetric editorial layouts; oversized display type
- All numbers animate scoreboard-style (odometer/flip)
- Custom easing everywhere: cubic-bezier(0.16, 1, 0.3, 1)
- Marketing pages: cinematic (R3F/GSAP). 
  Booking flow: fast, clean, ticket-stub metaphor.
- Every page ships with one signature interaction
